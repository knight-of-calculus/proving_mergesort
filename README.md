# proving_mergesort
Формально верифицированный алгоритм сортировки слиянием (в варианте "bottom-up"). Выступление на TechDay ВСК 01/11/2025

Программа практической части:
1. реализация mergesort без верификации; сравнение кода на Python и Idris
2. формализация свойств перестановки и сортировки
3. формализация понятия сортируемости (интерфейс для входных данных)
4. формальная спецификация для mergesort
5. верификация алгоритма
6. альтернативная реализация mergeSingletons и доказательство ее тотальности
7. обеспечение совместимости алгоритма с различными типами данных (доказательства удовлетворения данных интерфейсу)
8. доказательство алгоритмической сложности
9. доказательство стабильности

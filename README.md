# proving_mergesort
Формально верифицированный алгоритм сортировки слиянием (в варианте "bottom-up"). Выступление на TechDay ВСК 01/11/2025

Программа практической части:
1. - [x] реализация mergesort без верификации; сравнение кода на Python и Idris
2. - [x] формализация свойств перестановки и сортировки
3. - [x] формализация понятий порядка и сортируемости (интерфейс для входных данных)
4. - [x] формальная спецификация для mergesort
5. - [x] верификация алгоритма
6. - [x] альтернативная реализация mergeSingletons и доказательство ее тотальности
7. - [ ] обеспечение совместимости алгоритма с неиндуктивными типами данных (доказательства удовлетворения примитивных типов данных интерфейсу)
8. - [ ] доказательство алгоритмической сложности
9. - [ ] доказательство стабильности
